# Chinese Chess Game

Build and Run
-------------

To build the project:

    % make build

To start the Server:

    % make run-server

To start the High Concurrency Test:

    % make run-test

To start the Client:

    % make run-client

Default:

* TCP Port: "8889"

* Server IP: "127.0.0.1"
